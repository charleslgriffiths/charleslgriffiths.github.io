SenseFit (COMP6013)

Developing SenseFit – a machine learning-powered mobile application designed to predict injury risks for amateur and semi-professional athletes. Unlike existing solutions tailored for elite sports, SenseFit focuses on accessibility and usability. The app analyzes key biometric data such as heart rate variability (HRV), sleep quality, and fatigue levels to provide personalized injury risk assessments and proactive training recommendations. It integrates advanced ML models like Random Forest, k-Nearest Neighbors (kNN), and Long Short-Term Memory (LSTM) networks to detect trends and optimize injury prevention strategies. Built with React Native, the application ensures a seamless user experience while prioritizing data security and GDPR compliance.

Skills: Python, React Native, JavaScript, Agile Project Management, Presentation Skills, Expo

Machine Learning Model Evaluation & Neural Network Implementation (COMP6011)

For the COMP6011 Machine Learning module, I undertook two key projects using the Palmer Penguins dataset. In the first part, I applied and compared Naïve Bayes and k-Nearest Neighbors (kNN) classifiers by performing data preprocessing, cross-validation, hyperparameter tuning, and performance analysis to identify the best classification model. In the second part, I implemented a neural network from scratch in R, beginning with a single-layer perceptron and extending it into a multi-layer perceptron (MLP) with forward propagation and backpropagation. Hyperparameters such as learning rate, number of epochs, and training size were carefully tuned, and the model’s generalization was evaluated using validation techniques and error analysis.

Skills: Machine Learning, Neural Networks, Backpropagation, Hyperparameter Tuning, R

Robo-Uber: AI-Based Autonomous Taxi Dispatch System (COMP6032)

As part of the COMP6032 Artificial Intelligence module, I developed and optimized an AI-driven autonomous taxi dispatch system named Robo-Uber. I improved upon a given baseline simulation by implementing A* search for efficient path planning and integrating a probabilistic traffic-aware routing system. Additionally, I enhanced the dispatcher’s fare allocation strategy to maximize earnings while ensuring fair taxi utilization. The bidding system was optimized through profitability-based decision-making that considered traffic conditions, fare expiry, and expected returns. This project provided hands-on experience in heuristic search algorithms, probabilistic modelling, and real-world transportation system optimization.

Skills: Artificial Intelligence, Heuristic Analysis, Python, Probabilistic Models, Path Planning

Advanced Data Structures & Pathfinding Algorithms in Java (COMP5017)

In the Data Structures and Algorithms module, I implemented key data structures and graph traversal algorithms to optimize data retrieval and pathfinding. This project involved creating a hash table with custom quadratic probing for efficient storage, a binary search tree (BST) for O(log n) lookups, and applying various graph traversal techniques—including Depth-First Search (DFS), Breadth-First Search (BFS), Dijkstra’s Algorithm, and A* search—to determine optimal paths in a simulated transport network. Performance logging, assertions, and test-driven development (TDD) were employed to compare algorithm efficiency and refine the implementations.

Skills: Java, Data Structures, Pathfinding Algorithms, Graph Traversal
